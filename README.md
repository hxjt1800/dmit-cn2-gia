## 为什么电信用户买香港VPS要盯着CN2 GIA不放

先说清楚一个概念，很多人买VPS时被"CN2"这个词糊弄过。

**CN2分两种：**

- **CN2 GT（Global Transit）**：骨干节点是CN2，但接入层和末端可能走普通163网络，高峰期容易拥堵
- **CN2 GIA（Global Internet Access）**：全程CN2专属通道，去程回程都走高优先级路由，晚高峰表现明显更稳

电信用户从大陆访问香港，走CN2 GIA的延迟通常在30–40ms区间，这个数字在香港VPS里算是天花板级别。DMIT香港机房的电信线路，去程和回程都是CN2 GIA，这是它在电信用户圈子里口碑一直不错的核心原因。

联通用户走的是AS929或AS4837，移动用户走CMI，DMIT对三网都有针对性的路由优化，但本文重点放在电信CN2 GIA这条线上。
